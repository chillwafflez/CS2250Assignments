Anime Reminder Application

Personal project utilizing Python whereupon running, checks my AniList profile for animes I am currently watching using the AniList API. Anime are Japanese animated shows and AniList is an online platform for tracking anime show progression. After obtaining shows I'm currently watching, it sets reminders for the next episode of each series that airs that day.
